# 📊 Dynamic Programming (Phase 3 - Python)

Dynamic Programming (DP) is used to solve problems by breaking them into smaller overlapping subproblems and storing results.

---

## 📌 Topics Covered

- Fibonacci (Memoization & Tabulation)
- Climbing Stairs
- Coin Change (Minimum Coins)
- 0/1 Knapsack
- Longest Common Subsequence (LCS)
- Longest Increasing Subsequence (LIS)

---

## 🎯 Key Learnings

- Memoization vs Tabulation
- State definition
- Transition logic
- Optimization of recursive solutions
- Time & space complexity reduction
