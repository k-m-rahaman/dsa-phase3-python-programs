# 🔁 Backtracking (Phase 3 - Python)

Backtracking is a problem-solving technique where we try all possible solutions and undo choices when needed.

It is widely used in recursion-based problems and is very important for coding interviews.

- Permutations
- N-Queens Problem
- Rat in a Maze
- Combination Sum

## 🎯 Key Learnings

- Recursion + Decision Tree
- Choosing and un-choosing (backtrack step)

- Constraint-based problem solving
